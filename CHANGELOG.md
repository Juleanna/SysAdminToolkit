# Changelog

Усі значущі зміни проекту документуються у цьому файлі.

Формат базується на [Keep a Changelog](https://keepachangelog.com/uk/1.1.0/),
проект дотримується [Semantic Versioning](https://semver.org/lang/uk/).

## [5.0.0] -- 2026-02-25

### Додано

- **WPF GUI з темою Catppuccin Mocha** -- повноцінний графічний інтерфейс з темною темою, пошуком, категоріями, логом та історією виконання
- **18 категорій скриптів** -- Інвентаризація, Моніторинг, Сервіси, Диски, Сертифікати, Заплановані завдання, Бекапи, Мережа, Принтери, Профілі, Масові операції, Безпека, Відновлення, Утиліти, Telegram, Віддалена допомога, Active Directory, Звіти
- **67+ PowerShell-скриптів** для типових задач адміністрування
- **Модуль ToolkitCommon.psm1** -- 19 експортованих функцій (логування, звіти, retry, RBAC, email, сповіщення, DPAPI credentials)
- **Telegram-інтеграція** -- відправка повідомлень, алертів з рівнями критичності, файлів та інвентаризації
- **HTML-звіти** з Catppuccin-стилізацією та можливістю генерації дашбордів
- **RBAC-система** -- три ролі (Admin, Operator, Auditor) через `Config/Roles.json`
- **Windows Event Log** -- аудит-лог з джерелом SysAdminToolkit
- **Гарячі клавіші GUI** -- F5 (виконати), Esc (скасувати), F1 (довідка)
- **Пошук операцій** у GUI з фільтрацією за назвою та описом
- **Історія виконання** -- вкладка DataGrid з часом, дією, статусом та тривалістю
- **Прогрес-бар** -- підтримка маркерів `PROGRESS: N` у виводі скриптів
- **Конфігурація** -- `ToolkitConfig.json` з дефолтними значеннями та валідацією
- **Hosts.json** -- централізований список керованих ПК для масових операцій
- **Telegram.json** -- конфіг бота з підтримкою змінної оточення `SYSADMINTK_BOTTOKEN`
- **Логування з ротацією** -- `Write-TkLog` з рівнями DEBUG/INFO/WARN/ERROR та автоматичною ротацією за розміром
- **Retry-механізм** -- `Invoke-WithRetry` та `Invoke-TkRemote` з автоматичними повторними спробами
- **DPAPI-збереження облікових даних** -- `Get-TkCredential` для безпечного кешування credentials
- **WPF Balloon-сповіщення** -- `Show-TkNotification` для системного трею
- **Перевірка передумов** -- `Test-TkPrerequisite` (Admin, Modules, PS version, Online)
- **Кольоровий вивід** -- автоматична підсвітка помилок/попереджень/успіхів у GUI-лозі
- **Подвійний клік** для запуску операції зі списку
- **Експорт GUI-логу** у текстовий файл у каталог Reports
- **Каталог Tests/** для Pester-тестів
- **PSScriptAnalyzer конфігурація** -- `.PSScriptAnalyzerSettings.psd1`
- **Install.ps1** -- інсталятор з перевірками, створенням директорій та ярликів
- **Update-Toolkit.ps1** -- оновлення через Git або ZIP
- **Sign-Scripts.ps1** -- масовий підпис скриптів сертифікатом CodeSigning

### Безпека

- 19 скриптів у категорії Безпека: USB-блокування, Firewall, RDP/NLA, BitLocker, Defender, аудити, baseline export, LSA/SmartScreen та інші
- Облікові дані ніколи не зберігаються у відкритому вигляді (DPAPI або змінні оточення)
- RBAC обмежує доступ до категорій залежно від ролі користувача
